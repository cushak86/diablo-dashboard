Vercel을 통해 배포한 웹사이트가 네이버나 구글 등에서 검색되지 않는 이유는 크게 **검색엔진의 인지도 부족, 도메인 권위, 그리고 SEO(검색엔진 최적화) 설정** 때문입니다. `f1f4.net`처럼 검색이 잘 되게 하려면 검색엔진 봇(Bot)이 사이트를 발견하고 내용을 읽어갈 수 있도록 직접 길을 열어주어야 합니다.

현재 검색이 안 되는 주요 이유와 이를 해결하기 위한 구체적인 방법은 다음과 같습니다.

### 1. 검색엔진 웹마스터 도구에 직접 등록하지 않은 경우 (가장 큰 원인)

단순히 Vercel에 사이트를 배포했다고 해서 네이버나 구글이 사이트의 존재를 자동으로 알아채지는 못합니다. 검색 로봇이 사이트를 방문하게 하려면 수동으로 사이트를 알려주어야 합니다.

* **해결 방법:**
* 구글 서치 콘솔(Google Search Console)과 네이버 서치어드바이저(Naver Search Advisor)에 접속하여 `[https://diablo-dashboard-phi.vercel.app](https://diablo-dashboard-phi.vercel.app)` 도메인을 등록하고 소유권 확인(HTML 태그 삽입 방식 등)을 진행하세요.
* 등록 후, 각 도구의 'URL 검사' 기능을 이용해 색인 생성을 수동으로 요청해야 합니다.



### 2. Sitemap 및 Robots.txt 누락

검색엔진 봇이 사이트에 들어왔을 때, 어떤 페이지들이 있는지(Sitemap)와 어떤 페이지를 읽어도 되는지(robots.txt)를 알려주는 지도가 필요합니다. 특히 `/terror-zone`과 같은 하위 경로까지 제대로 검색되려면 사이트맵이 필수적입니다.

* **해결 방법:**
* 프로젝트 최상위 경로에 `robots.txt`를 생성하고 검색 로봇의 접근을 허용(`Allow: /`)하세요.
* `sitemap.xml`을 생성하여 구글 서치 콘솔과 네이버 서치어드바이저에 제출하세요. (Next.js 등을 사용 중이라면 `next-sitemap` 라이브러리로 쉽게 자동화할 수 있습니다.)



### 3. Vercel 기본 서브도메인의 한계 및 백링크 부재

`.vercel.app`은 Vercel에서 제공하는 서브도메인입니다. 검색엔진은 `.com`, `.net`과 같은 독립적인 커스텀 도메인(Custom Domain)에 더 높은 신뢰도와 권위를 부여합니다. 또한, 만들어진 지 얼마 안 된 Vercel 링크는 외부에서 걸어준 링크(백링크)가 없기 때문에 로봇이 스스로 찾아오기 매우 어렵습니다.

* **해결 방법:**
* **커스텀 도메인 연결 (권장):** `f1f4.net`처럼 가비아 등에서 저렴한 도메인을 하나 구입하여 Vercel 프로젝트에 연결하는 것이 장기적인 검색 노출에 훨씬 유리합니다.
* **초기 백링크 구축:** 검색 봇은 링크를 타고 이동합니다. 운영 중이신 네이버 블로그 등에 이번에 만든 '디아블로 2 레저렉션 테러존 대시보드'를 소개하는 포스팅을 작성하면서 해당 URL을 링크로 남겨두세요. 검색 로봇이 블로그를 크롤링하다가 링크를 타고 웹사이트로 넘어가 훨씬 빠르게 색인을 시작하게 됩니다.



### 4. 렌더링 방식의 문제 (CSR vs SSR)

만약 해당 사이트가 순수 React나 Vue를 이용한 CSR(Client-Side Rendering) 방식으로 개발되었다면, HTML 소스코드에 알맹이가 비어있어 네이버 봇(Yeti)이 내용을 읽지 못할 확률이 큽니다. (구글 봇은 JS를 실행해서 읽어내지만, 네이버는 이에 다소 취약합니다.)

* **해결 방법:**
* `<head>` 영역에 메타 태그를 충실히 작성해야 합니다. `<title>`, `<meta name="description">`에 '디아블로2 레저렉션 공포의 영역 추적기', 'D2R 테러존 시간표' 등 사용자들이 실제로 검색할 만한 핵심 키워드를 명확하게 포함하세요.
* 링크 공유 시 예쁘게 보이도록 Open Graph (`og:title`, `og:description`, `og:image`) 태그도 꼼꼼히 세팅해 두는 것이 좋습니다.



우선 **구글 서치 콘솔과 네이버 서치어드바이저 등록**을 1순위로 진행하시고, 보유하신 **블로그를 통한 링크 노출**을 병행하시면 며칠 내로 검색 결과에 반영되기 시작할 것입니다.
