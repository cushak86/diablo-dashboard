디아블로2 레저렉션 공포의 영역(테러존) 실시간 확인 사이트를 만들었습니다
디아2 공포의 영역 어디인지 매번 찾기 귀찮아서 직접 만든 대시보드
D2R 테러존 시간표 + 우버디아 + 트레더리 한글 검색을 한 곳에서

[본문 - 여기서부터 복사]

디아블로2 레저렉션 하면서 다들 한 번쯤 겪어보셨을 겁니다.

"지금 공포의 영역 어디지?"

게임에 들어가서 확인하거나, 영어로 된 해외 사이트를 뒤지거나...
매번 이게 번거로워서, 공포의 영역 확인에 필요한 것들을
한 화면에 모은 사이트를 직접 만들었습니다.

무료이고, 설치 없이 브라우저에서 바로 쓸 수 있습니다.

https://diablo-dashboard-phi.vercel.app

(위 링크를 붙여넣으면 링크 카드가 자동 생성됩니다)


■ 1. 공포의 영역 실시간 확인 — 다음 지역까지 미리

▶▶ 이미지 삽입: 01-terror-zone.png ◀◀

접속하면 바로 지금 공포의 영역이 어디인지,
그리고 다음 정각에 어디로 바뀌는지가 보입니다.
남은 시간은 큼직한 카운트다운으로 표시되고요.

지역 이름은 전부 한국 클라이언트 공식 명칭입니다.
"증오의 억류지", "비전의 성역", "약탈자 밀림"처럼
실제 게임에서 보는 그 이름 그대로라서
영어 사이트 보며 "이게 어디더라..." 할 필요가 없습니다.


■ 2. 음성 알림 — 파밍하다가 놓치지 않게

개인적으로 제일 유용하게 쓰는 기능입니다.
우측 상단 [소리 켜기]를 눌러두면,

· 변경 10분 · 5분 · 1분 전 → 알림음 + 한국어 음성으로 다음 지역을 읽어줍니다
· 정각 변경 시 → 차임벨 + 변경 안내 멘트

모니터 한쪽이나 폰에 띄워두고 게임하다 보면
소리만 듣고도 "아, 다음은 트라빈칼이구나" 하고 준비할 수 있습니다.


■ 3. 우버 디아블로(클론 디아) 진행도

▶▶ 이미지 삽입: 02-uber-diablo.png ◀◀

공역 페이지 하단에는 우버 디아블로 진행도가 함께 나옵니다.
래더/논래더, 소프트코어/하드코어, 아시아·미국·유럽
지역별 6단계 진행도를 확인할 수 있어서
멸망파괴반(안니 파밍) 준비할 때 유용합니다.


■ 4. 신규 아이템 트레더리 검색 — 한글로 치면 영문명이 나옵니다

트레더리(Traderie)에서 거래해보신 분들은 아실 텐데,
악마술사의 군림 신규 아이템은 한글 검색이 안 됩니다.
"공허가 영어로 뭐더라?" 하고 매번 찾아보게 되죠.

그래서 신규 아이템 전용 검색 탭을 만들었습니다.

▶▶ 이미지 삽입: 04-new-items-search.png ◀◀

· 신규 룬워드 · 고유 · 세트 · 파괴 부적 · 마법서 베이스 등 전부 수록
· 한글명 · 초성(ㄱㅎ) · 영문명 어느 쪽으로든 검색 가능
· 영문명 클릭 한 번으로 복사 → 트레더리에 붙여넣기
· [트레더리] 버튼으로 해당 아이템 거래 페이지 바로 이동

"잠복"만 쳐도 잠복하는 부적 시리즈가 전부 나옵니다.


■ 5. 모바일에서도 잘 보입니다

▶▶ 이미지 삽입: 06-mobile.png ◀◀

폰에 띄워두고 소리 켜두면 훌륭한 공역 알리미가 됩니다.


■ 앞으로 추가할 것들

· 룬워드 페이지 — 조합 순서 정확도에 신경 쓴 표 (준비 중)
· 프레임 기준(FCR/FHR) 페이지 — 직업별 구간표 (준비 중)

실시간 데이터는 해외 커뮤니티 트래커(d2runewizard 등) 기반이고,
개인이 취미로 운영하는 무료 사이트입니다.

쓰시다가 불편한 점이나 추가됐으면 하는 기능이 있으면
댓글로 알려주세요. 사이트가 도움이 되셨다면
사이트 우측 상단의 [후원하기]로 응원해주실 수도 있습니다 :)
